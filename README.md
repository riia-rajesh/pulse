## Daily Briefing Bot

| Section | Details |
| --- | --- |
| **Problem** | Wanted a daily briefing without doing anything manually each morning |
| **Approach** | Python bot with four functions pulling two free APIs, scheduled on GitHub Actions |
| **Craft** | Graceful try/except on every call, secrets handled properly, downloadable artifact |
| **Outcome** | Runs autonomously at 8 AM IST every day, for free. [Repo link](https://github.com/riia-rajesh/pulse) |
